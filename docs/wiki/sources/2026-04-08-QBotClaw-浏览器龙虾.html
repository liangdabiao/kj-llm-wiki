2026-04-08 QBotClaw 浏览器龙虾

腾讯云发布国内首个浏览器"龙虾"QBotClaw，通过微信Clawbot扫码直连，实现一句话指挥QQ浏览器自动完成复杂任务。

核心观点

  1. QBotClaw 是国内首个浏览器"龙虾"，由腾讯云发布，一句话就能指挥QQ浏览器自动干活
  2. 微信直连能力：支持微信Clawbot扫码直连，可远程指挥电脑干活
  3. 四大核心能力：文件操作、执行命令、浏览器自动化、信息搜索
  4. 兼容 OpenClaw 技能，支持自由配置国内各大主流大模型的API Key
  5. 安全机制：安全沙箱隔离、指令Markdown约束、SkillHub认证机制、黑名单机制

关键概念

  • QBotClaw — 腾讯云发布的浏览器"龙虾"，基于QQ浏览器的AI智能体
  • 浏览器龙虾 — 浏览器+AI智能体的产品形态，能看懂网页元素、懂上下文
  • 微信Clawbot — 微信端的AI机器人，扫码即可连接QBotClaw远程指挥电脑
  • OpenClaw — QBotClaw兼容的技能系统
  • QQ浏览器 Skill — 腾讯自研的浏览器Skill，能看懂动态网页元素
  • 端侧智能体 — 运行在用户设备端的AI智能体，非云端

应用场景

1. 自动比价

  • 指令示例："帮我对比Mac mini在淘宝、京东的价格、优惠、发货地，生成对比表"
  • 执行流程：自动打开电商页面 → 搜索关键词 → 对比生成表格
  • 限制：需要用户手动登录电商平台账号

2. 多平台发内容

  • 指令示例："把当前文档的内容配上图，发一条微博"
  • 执行流程：搜索引擎搜集资料 → 输入文本 → 发布微博

3. 微信远程指挥电脑

  • 用户在微信端输入指令，可指挥办公室电脑完成复杂自动化流程

与其他素材的关联

  • 与 agent-browser 概念相关：都是浏览器自动化能力的体现
  • 与 端侧智能体 趋势相关：代表AI工具向本地化、轻量化方向发展
  • 与 微信生态 整合：微信作为入口连接各种AI工具
