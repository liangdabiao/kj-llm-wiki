Cron 定时任务

OpenClaw的精确调度机制，在指定时间执行指定任务。

基本信息

  • 类型：系统机制
  • 触发方式：cron表达式/固定间隔
  • 作用：定期自动执行任务（早报、数据抓取、反思总结等）

核心作用

Cron 专注于"在何时执行什么"：

  • 精确时间调度
  • 独立会话执行
  • 可覆盖模型配置

使用场景

每日任务

  • 每天早上8点自动生成AI早报
  • 每天下午6点自动汇总业务数据
  • 每天晚上22:00自动反思总结

定期抓取

  • 每天早晚2次抓取竞品价格
  • 每周自动生成选品雷达
  • 每小时检查物流状态

监控预警

  • 每4小时监控广告ACoS
  • 每2小时巡检店铺状态
  • 实时检查客服回复

Cron vs Heartbeat

维度 Cron Heartbeat
核心价值 精确时间、独立会话 知道何时不该打扰
触发方式 cron表达式 固定间隔30m+主动唤醒
会话影响 可创建临时会话 恢复updatedAt防止保持活跃
错误处理 错退避、自动禁用 跳过、记录事件

配置示例

# 每天早上8点生成AI早报
0 8 * * *

# 每天晚上10点反思总结
0 22 * * *

# 每4小时监控广告ACoS
0 */4 * * *
