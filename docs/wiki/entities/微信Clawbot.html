微信Clawbot

微信端的AI机器人，扫码即可连接QBotClaw等浏览器龙虾，实现远程指挥电脑。

基本信息

  • 产品形态：微信AI机器人
  • 功能：作为微信与电脑端AI智能体的连接桥梁
  • 首次出现：2026年4月8日，与QBotClaw一同发布

核心能力

扫码直连

  • 用户用微信扫描QBotClaw展示的二维码
  • 建立微信与QQ浏览器的连接
  • 连接后可在微信端输入指令

远程指挥

  • 在微信输入指令
  • 驱动办公室电脑执行任务
  • 支持复杂的自动化流程

使用场景

场景：办公室远程操作

  1. 在电脑端打开QQ浏览器和QBotClaw
  2. 用微信扫描QBotClaw的二维码
  3. 在微信端输入指令（如"帮我对比某商品在淘宝京东的价格"）
  4. QBotClaw在电脑上自动执行

场景：随时随地驱动电脑

  • 不需要坐在电脑前
  • 通过微信即可远程驱动电脑
  • 适合执行耗时任务

不同素材中的观点

  • 验证了微信作为AI工具入口的可行性
  • 代表轻量化端侧智能体+微信入口的产品方向
