学习路径：从零基础到高阶
推荐学习顺序
第一阶段：建立工作规范（30-60 分钟）
编写 AGENTS.md 工作手册
设计 session 启动流程
制定记忆写入规范
设置安全边界和权限
第二阶段：优化记忆系统（60-120 分钟）
开启 memoryFlush 防止信息丢失
改进日志格式提高检索准确率
配置自动记忆维护机制
调整 embedding 模型参数
第三阶段：应用高级功能（120-240 分钟）
部署子 Agent 实现任务分发
创建 Cron 定时任务
开发自定义 Skill 扩展功能
配置多渠道接入方案
第四阶段：性能调优（1-2 天）
微调模型参数
优化 token 使用效率
配置缓存策略
建立性能监控体系
学习建议
按部就班：不要跳过基础环节，每个配置都有其存在的价值
边学边练：每完成一个配置，立即进行测试验证
问题记录：遇到问题及时记录，便于后续排查和解决
配置备份：重要修改前务必备份配置文件
