  整体架构

  该 CRM 系统包含 6 张数据表，覆盖了从客户管理到合同签约的完整销售流程：

  客户管理 ──→ 商机管理 ──→ 合同管理
     │              │
     └── 客户跟进记录 ←─┘

  销售人员管理（独立维度）
  💡使用说明书（操作指引）

  ---
  1. 客户管理（18 个字段，20+ 条记录）

  ┌────────────────┬──────────┬───────────────────────────────────────────────────┐
  │      字段      │   类型   │                       说明                        │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 客户名称       │ 文本     │ 主标识                                            │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 行业信息       │ 单选     │ 电商/互联网/金融/医疗/餐饮/科技/制造              │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 客户规模       │ 单选     │ <10 / 10-100 / 100-1000 / 1000-10000 / >10000     │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 所在城市       │ 文本     │ 北京/上海/深圳/成都等                             │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 客户地址       │ 地理位置 │ 精确经纬度定位                                    │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 对接人姓名     │ 文本     │ 客户方对接人                                      │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 对接人职位     │ 单选     │ CEO/CTO/IT负责人/人事负责人/产品负责人/运营负责人 │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 对接人邮箱     │ 文本     │ 邮箱地址                                          │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 联系电话       │ 数字     │ 联系方式                                          │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 客户所有人     │ 人员     │ 负责的销售人员                                    │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 销售leader     │ 查找引用 │ 从销售人员管理表自动关联                          │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 创建时间       │ 日期     │ 记录创建时间                                      │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 是否为公海客户 │ 公式     │ 自动判断                                          │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 客户名称查重   │ 公式     │ 自动检测重复客户                                  │
  ├────────────────┼──────────┼───────────────────────────────────────────────────┤
  │ 最近跟进时间   │ 查找引用 │ 从跟进记录自动关联                                │
  └────────────────┴──────────┴───────────────────────────────────────────────────┘

  亮点：
  - 自带客户查重功能（公式字段自动检测重复名称）
  - 支持公海客户判断（无人认领的客户进入公海）
  - 通过地理位置字段定位客户位置

  已录入客户示例：海雅卡特集团（成都/电商）、国信易有限公司（北京/互联网）、意昂集团（上海/电商）、大都人寿（深圳/金融）
  、EXOR集团（上海/医疗）等

  2. 商机管理（14 个字段，10 条商机）

  ┌───────────────────┬──────────┬─────────────────────────────────────────────────┐
  │       字段        │   类型   │                      说明                       │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 客户名称          │ 关联     │ 关联到「客户管理」表                            │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 跟进阶段          │ 单选     │ 待接触 / 初步提议 / 协商议价 / 丢失客户 / 赢单  │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 业务价值          │ 数字     │ 预估商机金额                                    │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 商机描述          │ 文本     │ 商机详情                                        │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 预计交易日期      │ 日期     │ 预计签约时间                                    │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 跟进销售人员      │ 人员     │ 负责销售                                        │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 销售leader        │ 查找引用 │ 自动关联                                        │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 销售区域          │ 查找引用 │ 从销售人员表自动关联                            │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 机会名称          │ 公式     │ 自动拼接生成（如"国信易有限公司-协商议价-..."） │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 丢单原因          │ 多选     │ 超客户预算等                                    │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 是否已有合同      │ 查找引用 │ 从合同管理表自动关联                            │
  ├───────────────────┼──────────┼─────────────────────────────────────────────────┤
  │ 合同管理-商机名称 │ 关联     │ 关联到「合同管理」表                            │
  └───────────────────┴──────────┴─────────────────────────────────────────────────┘

  商机阶段分布：
  - 赢单：3 笔（安盛科技、中星通讯、EXOR集团）
  - 协商议价：1 笔（国信易）
  - 初步提议：2 笔（意昂集团、优尔有限公司）
  - 待接触：2 笔（同用汽车、茂胜科技）
  - 丢失客户：2 笔（大都人寿、百草食品）

  业务价值范围：90,000 ~ 7,775,261
