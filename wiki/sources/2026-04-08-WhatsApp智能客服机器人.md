## 核心观点

1. **核心痛点**：时差导致响应延迟，白白流失订单
2. **OpenClaw三大能力**：多渠道Gateway、自动回复多语言支持、订单状态查询对接ERP
3. **硬件选择**：Mac mini M4 / 云服务器2核4G / Windows Linux PC 8GB以上
4. **配置流程**：安装OpenClaw → 配对WhatsApp → 启动Gateway → 配置SOUL.md → 对接ERP API
5. **订单查询Skill**：创建SKILL.md + query_order.py脚本，调用ERP API查询
6. **成本估算**：AI客服7×24小时在线，成本仅为人工的1/50-1/100
7. **多语言支持**：现代大模型本身支持100+种语言，用户用什么语言提问就用什么语言回复
