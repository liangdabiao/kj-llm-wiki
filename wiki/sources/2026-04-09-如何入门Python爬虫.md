## 核心观点

1. **项目驱动学习**（谢科、段晓晨）——"入门是良好的动机，但作用缓慢。带着项目学，你会被目标驱动"。不要追求完美的"入门点"，直接上手做项目
2. **五步入门法**（经海斌）——Python 基础 → urllib/requests → re/BeautifulSoup → 多线程 → 数据库/文件操作，掌握这五步即可进行小规模数据抓取
3. **爬虫本质是图遍历**（谢科）——URL 队列 + 已访问集合，所有爬虫的 backbone 都是这个结构。效率瓶颈在于判重和带宽
4. **Bloom Filter 是大规模去重的关键**（谢科）——固定内存、O(1) 判重，100% 不漏爬，允许小概率重复爬。分布式爬虫中配合 Redis 使用
5. **不要怂就是干**（段晓晨）——遇到编码问题、登录问题、API 变化都是正常的，99% 的问题都能在网上找到解决方案
