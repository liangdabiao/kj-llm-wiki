### 1. 数据管理
- 创建、查询、编辑记录
- 管理字段、视图、数据表
- 条件格式高亮异常

### 2. 自动化操作
- 自动填写多维表格

- 创建自动化报表

## 跨境电商应用

### 选品数据库
| 字段 | 说明 |
|------|------|
| 商品名 | 产品名称 |
| 监控链接 | 竞品链接 |
| 实时销量 | 自动抓取 |
| 价格 | 当前售价 |
| 利润空间 | 自动计算 |
| 舆情评分 | AI分析 |

### 物流跟踪表
- 订单号、tracking number
- 物流商、当前状态
- 预计到达时间
- 异常标记（超7天无更新）

### 竞品监控表
- ASIN、当前价、历史趋势
- 建议售价、库存状态
- 价格变动警报
