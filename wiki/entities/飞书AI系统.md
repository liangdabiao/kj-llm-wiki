# 飞书AI系统

> 基于飞书+Claude部署的AI运营系统，13个专项助手覆盖跨境运营全链路。

## 基本信息

- **类型**：应用方案
- **核心工具**：Claude Code + claude-to-im + Sorftime MCP + 飞书Lark MCP
- **作用**：随时随地进入工作状态，手机发一条消息即可

---

## 系统架构

### 桥接部署
- 工具：claude-to-im（开源桥接方案）
- 配置：在Claude Code终端输入 `/claude-to-im setup`
- 效果：飞书给机器人发消息，Claude直接回复

### 13个助手覆盖范围
- 随手记录类：灵感收集整理
- 选品调研类：市场调研分析
- Listing分析类：13模块全维度诊断
- 广告优化类：广告报告分析
- 竞品监控类
- 利润核算类
- 采购需求书类
- 公众号内容助手类

---

## 核心能力

### 随手记录
丢一条短视频脚本或有启发的文章给飞书机器人，自动分类归档到多维表格。

### 选品调研
丢一个ASIN或关键词，自动调用Sorftime数据，3-5分钟出完整报告。

### Listing分析
13个模块全维度诊断，包含VOC差评分析、合规检查、广告关键词规划。
