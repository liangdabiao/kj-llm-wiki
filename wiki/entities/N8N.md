# N8N

> 开源工作流自动化工具，支持跨系统数据同步与清洗，是OpenClaw跨境电商自动化方案中的基础连接层。

## 概述

N8N（发音"nodemation"）是一款开源的工作流自动化平台，支持通过可视化界面编排自动化流程。在跨境电商场景中，N8N 常与 OpenClaw 配合使用，承担跨系统数据同步、API 对接、定时任务调度等角色。

## 与 OpenClaw 的关系

N8N 与 OpenClaw 在跨境电商自动化方案中形成互补：
- **OpenClaw** 负责 AI 推理、决策、内容生成
- **N8N** 负责系统间数据流转、定时触发、API 调度
- 两者通过 Skill 或 API 串联，实现端到端的自动化流程
