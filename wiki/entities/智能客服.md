> AI驱动的客服系统，实现7×24小时自动回复、工单处理、多语言支持。

## 基本信息

- **类型**：应用场景
- **核心能力**：意图识别、知识检索、自动回复、工单升级
- **作用**：降低客服成本97%+，响应时间从30分钟降至即时

## 核心能力

### 1. 自动回复
- 基于知识库（RAG）回答常见问题
- 95%+准确率处理FAQ
- 多语言自动识别和回复

### 2. 工单自动化
- AI无法解决时自动创建工单
- 智能升级给人工客服
- 情感分析触发紧急升级

### 3. 多渠道集成
- WhatsApp/Telegram/企业微信/飞书
- 一个知识库服务所有平台
- 跨渠道统一客户体验

## 搭建方案

### 基于OpenClaw的RAG客服
- **三阶段架构**：意图分析 → 知识检索 → 回答生成
- **技术栈**：千问/Claude + 本地JSON + 企业微信
- **成本**：月¥150 vs 人工¥5000+，节省97%
- **部署**：5分钟即开即用

### WhatsApp智能客服
- **硬件**：Mac mini M4 / 云服务器2核4G
- **配置**：OpenClaw Gateway + SOUL.md + ERP API
- **功能**：多语言回复、订单查询、自动欢迎语
- **成本**：AI客服成本为人工的1/50-1/100

## 升级规则

| 条件 | 动作 |
|------|------|
| 情感negative且confidence>0.8 | 立即转人工 |
| 不在知识库且重试>2次 | 创建工单 |
| 涉及价格/库存等敏感信息 | 先核实再回复 |
| 投诉/不满/愤怒 | 高优先级升级 |
